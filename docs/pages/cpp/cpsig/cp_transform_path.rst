transform_path
================

The functions ``transform_path_float`` and ``transform_path_double`` apply path transformations
to paths of input type ``float`` and ``double`` respectively.

   :content-only:

batch_transform_path
=======================

The functions ``batch_transform_path_float`` and ``batch_transform_path_double`` apply path
transformations to batches of paths of input type ``float`` and ``double`` respectively.
