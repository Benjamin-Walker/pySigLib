signature
============

The functions ``signature_float`` and ``signature_double`` compute the
signature of paths of input type ``float`` and ``double`` respectively.

.. doxygengroup:: signature_functions
   :content-only:

batch_signature
=================

The functions ``batch_signature_float`` and ``batch_signature_double`` compute a
batch of signatures of paths of input type ``float`` and ``double`` respectively.

.. doxygengroup:: batch_signature_functions
   :content-only:

sig_backprop
=================

The functions ``sig_backprop_float`` and ``sig_backprop_double``
backpropagate through the signature computation for paths of input
type ``float`` and ``double`` respectively.

.. doxygengroup:: sig_backprop_functions
   :content-only:

batch_sig_backprop
===================

The functions ``batch_sig_backprop_float`` and ``batch_sig_backprop_double``
backpropagate through a batch signature computation for paths of input
type ``float`` and ``double`` respectively.
