This directory contains some basic examples and timings compared
to existing packages such as `iisignature`, `signatory` and `esig`.

- `signature_example.py` - a simple example of signatures
- `sig_kernel_example.py` - a simple example of signature kernels
- `signature_times_plot.py` - plots elapsed time vs degree for signatures
- `sig_kernel_times_len.py` - plots elapsed time vs path length for signature kernels
- `sig_kernel_times_dim.py` - plots elapsed time vs path dimension for signature kernels
